# Stop Last But Not Least

A satirical website dedicated to the most overused corporate phrase of all time: "last but not least."

![Stop Last But Not Least](cat.jpg)

## About

Every time somebody says "last but not least," a small part of our soul dies. This website is a tribute to all those who have suffered through endless meetings, presentations, and speeches featuring this tired clich√©.

## Features

- **Random Snarky Quotes**: Click the button to endure another grumpy cat take on the phrase
- **Soul Crusher Counter**: Tracks how many times you've clicked (persisted in localStorage)
- **Smooth Animations**: Fade transitions for quotes and pulse effects for the counter
- **Interactive Cat Image**: Hover over the grumpy cat to reveal its full color with a glowing effect
- **Dark Theme**: Easy on the eyes during those long corporate meetings
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks, no build tools)
- CSS animations and transitions
- LocalStorage for persistence
- SVG favicon

## Design Features

- Multi-layered shadow effects for depth
- Smooth hover interactions
- Modern rounded corners
- Red accent color palette (#ff4d4d)
- Monospace typography for that tech-snarky vibe

## Run Locally

Simply open `index.html` in your browser. No server or build process required.

```bash
open index.html
```
